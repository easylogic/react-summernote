## Usage

The feature list provides an overview of all available editing tools and their functionalities. Each feature is represented by its name, the type of UI component it uses, and a brief description of its purpose.

## Component Types

- **select**: Dropdown menu for selecting options
- **button**: Clickable button to toggle or activate a feature
- **menu**: Dropdown menu with multiple options
- **dialog**: Opens a modal dialog for more complex interactions

## Additional Notes

This comprehensive list includes all editing features available in our editor. Some features may have similar functionalities but are designed for different use cases or offer slight variations in behavior.

The 'Component' column indicates the type of UI element used to access each feature, which can be helpful for understanding how users interact with these tools in the editor interface.
